property และ value คืออะไรแตกต่างกันอย่างไร property กำหนดคุณสมบัติของอิลิเม้น value กำหนดคุณสมบัตของ property
property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง
  1. Property for content : color , size , font , text, list-style-type, Height, Width, background
  2. property for border : border, table
  3. property for padding : padding (top right buttom left)
  4. property for margin : margin (top right buttom left)
  5. property for element : display, Height, Width, Position, Float, Align, Flex, Grid
value unit มีกี่ประเภทอะไรบ้าง แต่ละประเภทคืออะไร
  • Absolute lenghts
    1. cm เซนติเมตร
    2. mm มิลลิเมตร
    3. in นิ้ว
    4. px พิกเซล 1/96 นิ้ว
    5. pt พอยท์ 1/72 นิ้ว
    6. pc พิคาล 1/12 นิ้ว
  • relative lenghts
    1. em เท่าขนาดฟอนต์ของ element
    2. ex เท่าความสูงของฟอนต์ของ element
    3. vh เปอร์เซ็นต์ความสูงหน้าจอ
    4. vw เปอร์เซ็นต์ความกว้างหน้าจอ
    5. vmin เปอร์เซ็นต์ของ view point ต่ำสุด
    6. vmax เปอร์เซ็นต์ของ view point สูงสุด
    7. % เปอร์เซ็นต์ของ parent element
